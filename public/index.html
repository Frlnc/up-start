Up-Start

a simple setup to up-start web projects

Commands

clean

Cleans all assets and uploads that are in the configured assets and uploads directories.

scripts

Concats and minifies all the scripts that are in the configured scripts directory. There is a base module system pre-implemented for quick prototyping.

styles

Compiles all the style sheets that are in the styles directory. This supports less and sass and you must specify which to compile via the styles:less command or styles:sass command.

up-start

Helps scaffold your project via allowing the use of pre-defined sass or less setups (currently need a less implementation). Run up-start:sass or up-start:less.

Careful: up-start may overwrite existing files in the configured styles directory!

watch

Runs gulp.watch to allow for watching files as they're edited and fire off the appropriate gulp tasks to compile them. This also runs a livereload server.
